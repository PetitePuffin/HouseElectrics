# House Electric - Professional Electrical Services Website

A modern, responsive website for House Electric, a government-licensed electrical contracting company serving Delhi and NCR region.

## 🏗️ Project Overview

House Electric is a professional electrical services company that provides:
- 24/7 Emergency Electrical Repair Services
- Annual Maintenance Contracts (AMC)
- Air Conditioner Installation & Maintenance
- Installation, Testing & Commissioning (ITC) Projects
- Residential and Commercial Electrical Solutions

## 🚀 Getting Started

### Prerequisites
- Modern web browser
- Local web server (optional, for development)

### Installation
1. Clone or download the project files
2. Open `index.html` in your web browser
3. Navigate through the website

### Development Setup
1. Use a local web server (e.g., Live Server in VS Code)
2. Make changes to HTML, CSS, or JavaScript files
3. Refresh browser to see changes

## 🔧 Customization

### Colors
Update CSS variables in `:root` section:
```css
:root {
    --primary: #your-color;
    --secondary: #your-color;
    --accent: #your-color;
}
```
